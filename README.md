# Anis Alam · Interactive Personal Portfolio 🚀

A highly interactive, space-themed personal portfolio website built to showcase my projects and skills as a Data Analyst and Frontend Developer. The site features immersive WebGL background animations, cycling video layers, 3D CSS effects, and a custom cursor experience.

**🔗 Live Demo:** [Insert Link to Live Website Here]

## ✨ Key Features

* **Immersive Video Backgrounds:** Cycles seamlessly through high-quality space/tech-themed WebM/MP4 videos.
* **WebGL Particle Galaxy:** A lightweight, interactive 3D particle system built with raw WebGL that reacts to mouse movement.
* **Interactive UI Elements:**
    * Custom trailing cursor.
    * 3D parallax tilt effects on project cards.
    * Rotating 3D avatar cube.
    * Animated typewriter effect for the hero section.
    * Scroll-triggered entrance animations using the `IntersectionObserver` API.
* **Fully Responsive:** Optimized for desktops, tablets, and mobile devices.
* **Zero Dependencies:** Built entirely with plain HTML, CSS, and Vanilla JavaScript—no heavy frameworks or external libraries required.

## 🛠️ Tech Stack

* **Markup & Styling:** HTML5, CSS3 (CSS Variables, Flexbox, CSS Grid, 3D Transforms, Keyframe Animations)
* **Interactivity:** Vanilla JavaScript (ES6+), DOM Manipulation, Intersection Observer API
* **Graphics:** HTML5 `<canvas>`, WebGL Context

## 🚀 Local Setup & Installation

To run this project locally on your machine, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/Anis-alam/portfolio-website.git](https://github.com/Anis-alam/portfolio-website.git)
    cd portfolio-website
    ```

2.  **Add Video Assets:**
    Because GitHub limits large file uploads, you must ensure the following video files are placed in the root directory (the same folder as `index.html`). The website relies on these for the background:
    * `blackhole.mp4`
    * `hero-video.mp4`
    * `galaxy.mp4`

3.  **Run the Site:**
    Simply open `index.html` in your web browser. 
    *(Note: For the best experience and to prevent browser strict-CORS issues with WebGL or autoplaying videos, it is recommended to run a local development server. If you have VS Code, use the "Live Server" extension, or run `npx serve` or `python -m http.server` in the project directory).*
